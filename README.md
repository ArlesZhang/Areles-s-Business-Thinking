### 建立个人事业的关键认知

一、不是学会了才开始做，而是做了之后才能学会

二、学习不能直接致富，财富的积累是靠资源+认知+能力

资源：明白资源的重要意义，不要陷入仅学习不闭环的盲区
认知：学习能够提高认知，也能降低认知，选择正确的方向很重要
能力：学习能够提高自己的能力，但是不以为着资源也同样会增加
所以，正确的顺序是：                                             
学习-->提高能力-->获得好的职位/薪酬-->提升自己的阶级-->提高自己的认知/积累资源-->学会合作，交换认知&资源-->不断积累，从量变到质变！

### **社会地位、孤独感、时间焦虑——怎么面对？**

社会地位是存在的，人弱的时候确实容易被忽视。但要记住：

- **现在的不被重视，只是暂时的。** 你在积累真正的能力，而能力最终会变成你的底气。
- **孤独感是强者的常态。** 真正想往上走的人，都会经历一段“独自修炼”的时期。
- **时间焦虑是好事。** 它提醒你不要浪费生命，而不是让你恐惧。
